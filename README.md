# PYTHON-MINIMAL-KATA-TDD-TEMPLATE

This template can be used as starting point in basic TDD katas to be solved in python.

## Prerequisites
- Python >= 3.8
- Pytest
- Make (optional)
- A python virtual environment (optional)

## Quick start

### Run kata
`make run` or `python .`
### Run tests
`make test` or `python -m pytest`
